Preferences
###########


Settings
********
 
**Auto Reload**:
 When enabled material changes in the linked file will be automatically synced with the local file.
  :Default: ``Disabled``


Hotkeys
*******

**Material Picker**:
 Activates the :ref:`Material Pick` tool.
  :Default: ``Alt + M``

|
**Material Search**:
 Opens the :ref:`Material Search` menu.
  :Default: ``Alt + M``
